# 📊 Superstore Sales Dashboard (Tableau)

Interactive Tableau dashboard built to analyze sales performance, profitability, customer segments, and regional trends using the Tableau Sample Superstore dataset.

---

![Dashboard Demo](dashboard-demo.gif)

---

# 📌 Project Overview

This Tableau dashboard provides an interactive view of sales performance across products, customer segments, and geographic regions.

Users can dynamically filter data, compare metrics, and identify trends to support business decision-making.
